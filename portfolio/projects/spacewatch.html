Space Watch

TASK

For this assignment, we were told to come up with a concept for a made-up website and then show what it would look like on desktop and on mobile. In addition to this, we also had to design what the app icon might look like.

SOLUTION

My concept for this project was a space-themed website. On the site, the user would not only be able to gain information on each planet in our solar system, but they would also be able to see live footage of the planet from different cameras set up by NASA. The software we used to create this was Photoshop.
